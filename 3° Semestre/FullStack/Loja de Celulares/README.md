# 📱 Loja de Celulares - Projeto Full Stack

Projeto desenvolvido com **Node.js, MySQL e JavaScript**, com o objetivo de criar um sistema de cadastro e gerenciamento de celulares.

---

## 🚀 Funcionalidades

* Listar celulares
* Cadastrar novos
* Editar informações
* Excluir registros

---

## 🛠️ Tecnologias

* Node.js (Back-end)
* Express
* MySQL (XAMPP)
* HTML, CSS e JavaScript (Front-end)
* Insomnia (testes)

---

## ▶️ Como rodar o projeto

### 1. Banco de dados

* Inicie o XAMPP
* Crie o banco `loja_celulares`
* Crie a tabela `celulares`

---

### 2. Back-end

```bash
npm init
npm install express cors mysql2
npx nodemon server.js
node server.js
```

---

### 3. Front-end

Abra:

```bash
frontend/index.html
```
pelo navegador:

```
Open with Liver Server
```

---

## 🔗 Rotas da API

* GET /celulares
* GET /celulares/:id
* POST /celulares
* PUT /celulares/:id
* DELETE /celulares/:id
